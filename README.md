# ido4 Plugins Marketplace

Plugin marketplace for the ido4 ecosystem. Compatible with Claude Code and Cowork.

## Available Plugins

| Plugin | Description |
|--------|------------|
| [ido4shape](https://github.com/ido4-dev/ido4shape) | A thinking partner that helps crystallize what needs to be built |

## Installation

```bash
# Add the marketplace
claude plugin marketplace add ido4-dev/ido4-plugins

# Install a plugin
claude plugin install ido4shape@ido4-plugins
```
